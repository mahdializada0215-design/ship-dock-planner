📦 Pallet Building & Loading Tips

Building Pallets

  • Max height: 1.8m — no exceptions
  • Heavy boxes on BOTTOM, light on TOP
  • Interlock boxes — alternate direction each layer
  • Build pallets NOT staging — keep out of walkways
  • Wrap: 5 wraps at base, 3 in middle, 3 at top

Loading & Staging

  • Load based on CPT — earliest CPT loads first
  • Stage based on CPT — prioritise urgent departures
  • Keep dock clear at all times
  • If trucks late or TDR team late — let PA know immediately
  • Label all 4 sides — sort code visible from every angle

Water Spider

  • Supply pallet team — wrap, labels, carts
  • Move completed pallets to staging promptly
  • Balance WIP between paired lines
  • Flag PG if one line imbalanced
  • Ask team: need HC? Need pallet? Need pallet removed?

PG Checklist

  • Walk lines every 30 min — balance HC to volume
  • Track CPTs — know urgent departures
  • Escalate if vol > 95% capacity
  • Brief at start: target, trucks, priority lanes
  • End: flush rodeo, confirm all trucks closed

📖 How It Works

🔐 Login & Profiles

  • First time: enter your name + set a password
  • Your allocations are saved privately under your profile
  • Next time: enter the same name + password to load your data
  • Click the 👤 badge at the top to switch profiles or log out
  • Others can't see or edit your data without your password

📅 Managing Days

  • Click 📅 Today to set the current tab to today's date
  • Use the date picker to set any date for the active tab
  • Click + Add Day to add more day tabs
  • Hover a tab and click × to remove it
  • Double-click any tab label to rename it manually

👥 Allocating People

  • Click + on any line to add a person (Pallets)
  • Type names directly into W/S, Loaders, and FL fields
  • HC counts update live — only filled names are counted
  • Flex people (Pick/Pack/5S) are NOT counted in Total HC
  • Set the 🎯 Target (units) per day at the top

✏️ Editing & Customising

  • Click BWU6 in the title to change site name
  • Click PRIME to change event name (Peak, BAU, etc.)
  • All changes save automatically to your profile
  • Your data stays even if you close the browser
  • Different users on the same link get separate data
